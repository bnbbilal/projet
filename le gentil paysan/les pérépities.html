les pérépities

(La salle est archicomble .Les spectateurs discutent entre eux, se taisent quand ils entendent les trois coups qui marquent le début du spectacle .A ce moment-là, le rideau se lève sur une scène bien éclairée. Le décor est très simple, il se compose d’une maison, d’une grange et un mur qui sépare la ferme et la forêt. Les acteurs rentrent sur scène.)

Le filS DE PAYSAN:

mon père, mon père, vite, j’ai entendu les deux enfants, ils préparaient un plan pour voler les bananes, allez au champ pour garder les deux.


LE PAYSAN:

Ah !c’est vraiment dangereux !


BILAL:

viens, viens ici regarde le paysan qui va entrer dans les champs.


ILYASS:

allez, cueillie les bananes !


(Soudain ; le paysan les attrape).

BILAL:

: Laisse-moi, laisse-moi s’il te plaît!


ILYASS:

Nous voudrons juste des bananes.


LE PAYSAN:

Je vais t’emmener à la grange.


ILYASS:

: Qu’allons-nous faire maintenant !


BILAL:

: Nul doute qu’il nous donnera comme une nourriture pour ses chiens !ah, ah, ah !

(Le paysan les a emmenés à la grange, puis il les a attachés avec une corde).


(Après une période de conflit, les deux enfants sont calmés).


ILYASS:

Qu’est-ce qu’il écrit sur le mur ?


BILAL:

Je n’ai aucune idée.
